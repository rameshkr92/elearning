Reference

CSS Class Reference

General Helpers

.overflow-hidden - adds overflow: hidden on any element
.display-inline-block - adds display: inline-block to any element
.block - applies display: block to any element
.display-none - hides any element

Positioning Helpers

.relative - adds position: relative to any element
.absolute - applies position: absolute to any element
.top - applies top: 0 to any element - for use with .absolute
.bottom - applies bottom: 0 to any element - for use with .absolute
.left - applies left: 0 to any element - for use with .absolute
.right - applies right: 0 to any element - for use with .absolute

Sizing Helpers

.width-$size - applies a width of $size px to any element where $size can be any increment of 10 between 30 and 600
.width-$size-xs - same as width-$size except it only applies on extra small screens
.max-width-$size - applies a maximum width of $size px to any element where $size can be any increment of 10 between 30 and 600
.width-auto - applies width: auto to any element
.width-auto-$screen - applies width: auto to any element on various screen sizes where $screen can be any of xs, sm, md, lg
.width-100pc - applies width: 100% to any element
.width-100pc-$screen - applies width: 100% to any element on various screen sizes where $screen can be any of xs, sm, md, lg
.height-$size - applies a height of $size px to any element where $size can be any increment of 10 between 30 and 600
.height-$size-xs - same as height-$size except it only applies on extra small screens
.max-height-$size - applies a maximum height of $size px to any element where $size can be any increment of 10 between 30 and 600
.height-auto-$screen - applies height: auto to any element on various screen sizes where $screen can be any of xs, sm, md, lg

Spacing Helpers

.margin-v-$size - applies a margin top and bottom of $size px to any element where size can be anything between 1 and 15
.margin-v-$top-$bottom - same as margin-v-$size
.margin-none - disables margin on any element
.padding-none - disables padding on any element

Base Colour Helpers

.bg-primary - applies the color of @primary-color variable as background
.bg-default - applies #f6f6f6 as background and #808080 as text color
.bg-white - applies #fff as background
.bg-purple - applies the color of @purple-color variable as background
.bg-brown - applies the color of @brown-color variable as background
.bg-danger - applies the color of @danger-color variable as background
.bg-lightred - applies the color of @lightred-color variable as background
.bg-gray-dark - applies background-color #555 and color #fff
.bg-transparent - removes the background from any element

Sidebars

.sidebar - the base class required for any sidebar element - it applies directly on the sidebar element;
.left or .right - required for any sidebar element for positioning the sidebar to the left or to the right of the page.

Sidebar Sizes (All screen sizes)

.sidebar-size-1 - applies a width of 56px to the sidebar and hides all content within the sidebar except the main menu icons
.sidebar-size-2 - applies a width of 200px to the sidebar
.sidebar-size-3 - applies a width of 250px to the sidebar
.sidebar-offset-$size - required for any sidebar element; where $size can be any of the sidebar sizes (e.g. 1, 2 or 3); sets the position of the sidebar in relation to other sidebar elements and the rest of the layout and allows multiple sidebars to be placed in specific placements on each side; for example a sidebar element with .left and .sidebar-offset-2 will leave a gap for another .sidebar-size-2 element, starting from the left of the screen.

Sidebar Sizes (Specific screen sizes)

.sidebar-size-$screen-$size - where $size can be anything between 1 and 3 and $screen can be any of xs, sm, md, lg
.sidebar-offset-$screen-$size - required for any sidebar element using one of the $screen size classes above; $size can also be 0

Sidebar Visibility

Sidebars are not visible by default and such we must apply specific screen-size classes to enable their visibility.
.show-sidebar - must be applied directly on the <html> element in order to display any sidebars, on any screen size; this class is not only used for the sidebar visibility, but also updates the layout around sidebars in order to accommodate them.
.sidebar-visible-desktop - must be applied on the sidebar element in order to display the sidebar on desktop screens when .show-sidebar is applied to the <html> element;
.sidebar-visible-mobile - must be applied on the sidebar element in order to display the sidebar on mobile when .show-sidebar is applied to the <html> element;

Sidebar Page Layout

When using sidebars, we must let the rest of the layout know where the sidebars are positioned on the page in order to accommodate them. The following classes must be used in conjunction with the .show-sidebar class and must also be applied on the <html> element.
Single Sidebars on side. The following classes will be used when there are single sidebars on each side e.g. one sidebar on the left side and/or one sidebar on the right side. $size can be any valid sidebar size: 1, 2 or 3;
.sidebar-l$size - used for left sidebars on any screen size
.sidebar-r$size - used for right sidebars on any screen size
.sidebar-l$size-sm - used for left sidebars on small to medium screen sizes
.sidebar-r$size-sm - used for right sidebars on small to medium screen sizes
Multiple Sidebars on side. The following classes will be used when there are multiple sidebars on each side. $sizes is a number formed from concatenating any of the valid sidebar sizes (1, 2 or 3) used for the corresponding side, respecting the order of the sidebars from left to right. For example, when there are 2 sidebars on the left, first sidebar of size 1 and second sidebar of size 2, we will use .sidebar-l-sum-12. If on the same page, there is another sidebar on the right of size 2, we will also use the .sidebar-r2 class.
.sidebar-l-sum-$sizes - used for left sidebars on any screen size
.sidebar-r-sum-$sizes - used for right sidebars on any screen size
.sidebar-l-sum-sm-$sizes - used for left sidebars on small to medium screen sizes
.sidebar-r-sum-sm-$sizes - used for right sidebars on small to medium screen sizes
